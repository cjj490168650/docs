## Develop

```bash
git clone https://github.com/XCPCIO/XCPCIO.git -b main --depth=1

cd XCPCIO

# insall mkdocs-material and plugins
pip3 install -U -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple/

# install node deps
pnpm i

# build
pnpm build

# dev
pnpm dev
```
